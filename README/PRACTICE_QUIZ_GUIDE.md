## Troubleshooting

### "Questions not generating"
- Check that PDFs are uploaded
- Verify backend is running
- Try with fewer questions (5 instead of 20)

### "Can't click options"
- Make sure quiz isn't submitted yet
- If submitted, click "Try Again" to reset
- Refresh page if buttons are frozen

### "Submit button not working"
- Check if you answered at least one question
- Look for error messages
- Try clearing answers and re-answering

### "Score shows 0/0"
- This means no questions were answered
- Go back and select answers
- Make sure to click option buttons
